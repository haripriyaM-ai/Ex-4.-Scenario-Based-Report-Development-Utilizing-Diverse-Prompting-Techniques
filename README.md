Objective: The goal of this experiment is to design and develop an AI-powered chatbot that can handle customer inquiries, provide support, and improve customer experience in a retail environment. Create prompts using various AI prompting techniques to guide your experiment, data collection, analysis, and report creation.
## Aim: 
To design and develop an AI-powered chatbot for a retail environment using diverse prompting techniques that can handle customer inquiries, provide support, and improve customer experience.

## Algorithm: 
1. Identify key customer inquiry types in a retail environment.
2. Select appropriate AI prompting techniques — Zero-Shot, Few-Shot, Chain-of-Thought, Role-Based, Scenario-Based, and Constraint-Based.
3. Design prompts for each technique mapped to relevant retail scenarios.
4. Input the prompts into the AI model and collect the generated responses.
5. Evaluate the output based on:
   - Relevance and accuracy of responses
   - Tone and customer-friendliness
   - Handling of edge cases and escalations
6. Document the results and observations.
   
## Prompt:
